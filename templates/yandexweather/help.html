Информация

Модуль получения данных о погоде с сайта яндекс.

Для добавления своего города - нужно узнать из строки браузера номер своего города. [code]https://yandex.ru/pogoda/[b]2[/b] [/code] Например для Санкт-Петербурга это значение 2 нужно добавить в таблицу yaweather_cities на примере уже имеющихся данных и поставить 1 напротив поля check.

Обновления:
1) Добавлен прогноз погоды на день (утро/день/вечер/ночь)
2) Прогноз на 7 дней вынесен в отдельную вкладку.
3) Добавлен выбор избранного города, создается объект yw_mycity, куда будут копироваться свойства избранного города. Для выбора избранного города необходимо нажать "сердечко" на нужном.
yw_mycity избавит от необходимости править чужие сценарии, например для уведомления о погоде.
4) Добавлена возможность указать вместо id города его координаты. Необходимо в поле latlon таблицы yaweather_cities указать значение в формате lat=47.240585&lon=38.870989. В качестве id города можно указать любое значение, latlon будет в качестве более приоритетного параметра. Вместо наименование города спутника указывается имя большого города. Прогноз указан на город, соответсвующий коорданатам. Перед установкой обновления нужно полностью удалить плагин.
